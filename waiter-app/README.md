# Tanuki Waiter Helper

Веб-приложение для официантов, помогающее управлять заказами и просматривать меню.

## Функционал

-   **Галерея Меню**: Просмотр блюд с картинками и описанием.
-   **Управление Заказами**: Создание заказов по столам, чек-лист выноса блюд.
-   **Админка Меню**: Возможность добавлять фотографии и менять описание блюд.
-   **Командная работа**: Все официанты видят активные столы.

## Установка и Запуск

1.  **Установите зависимости**:
    ```bash
    npm install
    ```

2.  **Настройте Supabase**:
    -   Создайте проект на [Supabase](https://supabase.com).
    -   Скопируйте `URL` и `anon key` в файл `.env.local` (создайте его на основе `.env.example`).
    -   **Запустите SQL скрипты**:
        1.  `supabase/schema.sql` (Таблицы)
        2.  `supabase/storage.sql` (Хранилище для картинок)
    -   В Supabase Dashboard зайдите в **Storage**, создайте бакет `menu-images` и сделайте его Public (если скрипт не сработал).

3.  **Заполните Меню**:
    -   Я уже скачал список блюд (названия и цены) в файл `menu_data.json`.
    -   Запустите загрузку в базу:
        ```bash
        node scripts/seed_menu.mjs
        ```
    -   *Примечание:* Фотографии с сайта скачать не удалось из-за защиты. Вы можете добавить их вручную через раздел "Edit Menu" в приложении.

4.  **Запустите приложение**:
    ```bash
    npm run dev
    ```
    Откройте [http://localhost:3000](http://localhost:3000).

## Развертывание на Railway

1.  Загрузите код на GitHub.
2.  Зайдите на [Railway.app](https://railway.app) и создайте новый проект из GitHub репозитория.
3.  В настройках проекта на Railway добавьте переменные окружения:
    -   `NEXT_PUBLIC_SUPABASE_URL`
    -   `NEXT_PUBLIC_SUPABASE_ANON_KEY`
4.  Railway автоматически соберет и запустит сайт.
